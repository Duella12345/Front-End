Bookworm's Library


Welcome to Bookworm's Library.

Use the arrow keys (or 'wasd' keys) to control the Bookworm.
Move around the library, and gather up the golden book tokens, while avoiding the lava.
(Lava!? This library was clearly build in a poor location!)

Look out for the special purple Reading Glasses, which let you see secret shelves and walls.
Use the doors to move back and forth between rooms, or collect all the book tokens in a room to be teleported away to next room.
